## 30-second quick start

```bash
# 1. Install dependencies
npm install

# 2. Start development server (HMR on port 9000)
npm run dev

# 3. Run tests
npm run test:all
```

Open `http://localhost:9000` and click **Obter Localização** to begin.

See [`guides/GETTING_STARTED.md`](guides/GETTING_STARTED.md) for the
complete setup walkthrough, prerequisites, and troubleshooting.
