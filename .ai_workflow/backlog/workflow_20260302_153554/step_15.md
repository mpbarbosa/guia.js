### Issue 2: Color Contrast Violations
- **Category**: Accessibility/Visual
- **Severity**: Critical
- **Location**: src/test-distrito-bairro.html, src/test-error-handling.html, examples/address-converter.html
- **Description**: Some color combinations (e.g., #1976d2 on white, #c62828 on #ffcdd2) fail WCAG AA contrast requirements.
- **Impact**: Text may be unreadable for users with low vision or color blindness.
- **Recommendation**: Adjust color palette to ensure minimum 4.5:1 contrast ratio for text and interactive elements.

### Issue 3: Inconsistent Keyboard Navigation
- **Category**: Accessibility/Usability
- **Severity**: Critical
- **Location**: All files (no tabindex, focus management, or keyboard event handling detected)
- **Description**: Interactive elements (buttons, forms) lack explicit keyboard navigation support.
- **Impact**: Users relying on keyboard cannot efficiently interact with the UI.
- **Recommendation**: Implement tabindex, focus-visible styles, and ensure all controls are reachable and operable via keyboard.
